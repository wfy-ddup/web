javascript--function(函数)

函数

函数执行完return语句之后立即退出，因此，位于return语句之后的任何代码都永远不糊被执行

  function sum(num1,num2){
    alert("退出前");
    return num1 + num2;
    alert("退出后");              //永远不会被执行
  }

要么让函数始终返回一个值，要么始终不返回值。如果函数有时返回值，有时不返回值，会给调试代码带来不便。

严格模式，不能将函数或参数，命名为eval或arguments



函数的参数

①传入函数的参数不受函数所定义的参数的个数影响，可多可少

因为传入函数的参数在函数内部是一个数组来表示的,数组的length无所谓。实际上，在函数体内可以通过arguments对象来访问参数数组，从而获取每一个传入函数的参数
