javascript--switch语句



switch语句 与 if...else..语句

if...else..语句

  var i = 25;
    if(i === 25){
      alert("25");
    }else if(i === 35){
      alert("35");
    }else if(i === 45){
      alert("45");
    }else{
      alert("other");
    }

switch语句

  switch(i){
    case 25:
      alert("25");
      break;
    case 35:
      alert("35");
      break;
    case 45:
      alert("45");
      break;
    default:
      alert("other");
  }

switch语句中的break防止同时执行多个case代码

表达式--判断运算符一起使用--true/false

  var num = 25;
  switch(true){
    case num < 0:
      alert("Less than 0");
      break;
    case num >= 10 && num <=10:
      alert("between 0 and 10 ");
      break;
    case num > 10 &&num <= 20:
      alert("between 10 an 20");
      break;
    default:
      alert("More than 20");
  }

switch语句传递表达式true，是因为每个case值都可以返回一个布尔值
