bind,call,apply三者的区别

1.bind,call,apply三者都是改变函数this对象的指向
2.call传入的参数是枚举出来传入的
3.apply传入的参数是以一个数组的形式传入的
4.call,apply改变函数的this对象指向的同事并且立即执行函数
5.bind改变this指向时会创建一个新的函数并改了this指向，调用时正常的传入参数

          function isPrime(input) {
              if (input < 2) {
                  return false;
              } else {
                  for (var i = 2; i <= Math.sqrt(input); i++) {
                      if (input % i == 0) {
                          return false;
                      }
                  }
              }
              return true;
          }

closest()方法;

closest()方法获得匹配选择器的第一个祖先元素，从当前的元素开始沿DOM数向上。与parents()方法很像，但存在一点差异

closest():从当前元素开始，向上遍历，直到找到已应用选择器的零个，一个匹配为止

parents():从父元素开始，向上遍历，找到一个或者多个

          //closest()完成事件委派，当接近列表元素或其子代元素被点击时
          $(document).bind("click",function(e){
            $(e.target).closest('li').toggleClass('hilight');
          })
          //父元素变红
          <ul>
            <li class="item-a">A</li>
            <li class="item-b">B</li>
          </ul>
          $("li.item-a").closest('ul').css({'background':'red'});
